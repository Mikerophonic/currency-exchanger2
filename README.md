## Description

This application was created as my fifth independent project for Epicodus. It demonstrates my understanding of asychronous programming and making an API call.

## Setup/Installation Requirements

* Clone this repository to your local machine: $ git clone https://github.com/Mikerophonic/currency-exchanger.git
* To get an API key, go to Exchange Rate API site and click get free key (https://www.exchangerate-api.com/docs/overview). Sign up for free plan.
* create .env file in root directory and type 'API_KEY=' then paste your key.
* Install all packages with $ npm install.
* Build the project using webpack with $ npm run build
* Start a development server with $ npm run start
* Lint JS files in the src folder with $ npm run lint
